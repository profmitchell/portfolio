# Blog Posts Directory

Store blog posts as markdown files with frontmatter:

```md
---
title: "Post Title"
excerpt: "Brief description"
date: "YYYY-MM-DD"
author: "Mitchell Cohen"
category: "Category"
readTime: "X min read"
tags: ["tag1", "tag2"]
image: "https://..."
---

Content in markdown...
```

- Tutorial
- Gear
